
AISHA: AI-Powered Industrial Attachment Matching Platform
PROJECT PROPOSAL

Date: February 05, 2026
Version: 1.0

================================================================================
EXECUTIVE SUMMARY
================================================================================

The AISHA: AI-Powered Industrial Attachment Matching Platform is an intelligent, automated platform 
designed to revolutionize how students, educational institutions, and companies 
connect for industrial attachment opportunities. By leveraging AI/ML technologies 
and automation, this system eliminates manual processes, reduces corruption, and 
ensures optimal student-company matching based on performance, interests, and 
organizational needs.

================================================================================
1. PROBLEM STATEMENT
================================================================================

1.1 Current Challenges
----------------------

Students Face:
• Difficulty finding suitable attachment opportunities
• Manual application processes that are time-consuming
• Limited visibility into available positions
• Corruption and favoritism in placement processes
• Mismatch between skills/interests and placement
• Complex documentation requirements (letters, NITA forms, insurance, etc.)

Educational Institutions Face:
• Lack of centralized tracking of student placements
• Manual processing of attachment applications
• Difficulty monitoring student progress during attachment
• Limited data on placement success rates
• Administrative overhead in managing documentation

Companies/Organizations Face:
• Manual screening of numerous applications
• Difficulty finding students with specific skill sets
• Time-consuming placement process
• Lack of structured communication with institutions
• Administrative burden of managing attachments

1.2 Manual Process Inefficiencies
----------------------------------

Current attachment application involves:
• Application letters
• NITA forms (mandatory regulatory requirement)
• Insurance cover documentation
• School letters detailing units covered
• Manual submission to multiple organizations
• Follow-up communications
• Manual acceptance/rejection processes

14. LEARNING MODULE - SKILL DEVELOPMENT
================================================================================

14.1 Overview
-------------

In addition to attachment placement, the system includes an intelligent
Learning Module that helps students continuously improve their skills and
knowledge in their chosen field of interest.

14.2 Purpose
------------

The Learning Module addresses a critical gap: students need to continuously
develop their skills to remain competitive and excel in their attachments.
By providing personalized learning recommendations, the system ensures students
are well-prepared and continuously improving.

14.3 Key Features
-----------------

PERSONALIZED COURSE RECOMMENDATIONS:
• AI-powered recommendations based on student's field of interest
• Courses aligned with selected specialization (e.g., Software Dev, Networking)
• Difficulty level matched to current skill level
• Learning path progression tracking

CERTIFICATION GUIDANCE:
• Industry-recognized certifications recommended
• Certification roadmap based on career goals
• Free and paid certification options
• Exam preparation resources
• Certification tracking and badges

LEARNING PLATFORM INTEGRATION:
• Curated content from multiple platforms:
  - Coursera, edX, Udemy (MOOCs)
  - YouTube educational channels
  - FreeCodeCamp, Codecademy (coding)
  - LinkedIn Learning
  - Local Kenyan platforms
  - University open courseware

SKILL GAP ANALYSIS:
• Compare student's current skills with industry requirements
• Identify gaps based on attachment opportunities
• Recommend specific courses to fill gaps
• Track skill development over time

LEARNING ANALYTICS:
• Track courses started and completed
• Monitor learning progress
• Measure skill improvement
• Correlate learning with placement success

14.4 How It Works
-----------------

STEP 1: Interest Analysis
• Student selects field of interest during registration
• System analyzes selected specialization
• Identifies relevant skill domains

STEP 2: Skill Assessment
• Student completes optional skill assessment
• System analyzes academic units completed
• Identifies current knowledge level

STEP 3: Recommendation Generation
• AI matches student profile with learning resources
• Ranks recommendations by relevance
• Considers free vs. paid options
• Prioritizes beginner to advanced progression

STEP 4: Continuous Learning
• Student browses recommended courses
• Enrolls in courses of interest
• Tracks progress within the system
• Receives new recommendations as skills improve

STEP 5: Impact Measurement
• System tracks completed courses
• Updates student skill profile
• Improves attachment matching based on new skills
• Demonstrates ROI of learning efforts

• Language preferences

RANKING CRITERIA:

• Course quality and ratings (25%)
• Alignment with career goals (20%)
• Accessibility (free/affordable) (10%)
