STUDENT USER STORIES
================================================================================

EPIC 1: Student Registration & Profile Management
--------------------------------------------------

US-S1: As a student, I want to register using my institutional email so that
       my account is verified and linked to my institution.
Acceptance Criteria:
• Student can register with institutional email
• Verification email sent automatically
• Account activated upon email verification
• Student profile auto-populated from institution portal

US-S2: As a student, I want to complete my profile with my interests and
       specialization so that I get relevant attachment opportunities.
Acceptance Criteria:
• Student can select area of interest (e.g., Software Dev, Networking)
• Student can add skills and certifications
• Student can set location preferences
• Profile completion percentage displayed
• Cannot apply until profile is 100% complete

US-S3: As a student, I want to view my academic performance and units covered
       so that I can understand my qualifications.
Acceptance Criteria:
• GPA displayed prominently
• List of completed units with grades
• Units in progress shown separately
• Performance trends visualized
• Data synced from institution portal

Acceptance Criteria:

• Opportunities sorted by relevance
• Filter by location, company, department
• Search by keywords

US-S5: As a student, I want to see personalized recommendations so that I can
       quickly find the best matches for me.
Acceptance Criteria:
• Top 10 recommendations displayed on dashboard

• "Why this match" feature available

US-S6: As a student, I want to understand why an opportunity matches me so
       that I can make informed decisions.
Acceptance Criteria:
• Match reasons displayed (e.g., "Your GPA exceeds requirement")
• Skills match highlighted
• Location match indicated
• Department alignment shown
• Historical success rate for similar students
